# ポケモン図鑑 Webアプリケーション

ポケモンデータを表示するNext.jsベースのWebアプリケーションです。

## 機能

- ポケモン一覧表示（第1世代151匹）
- ポケモン詳細表示
- レスポンシブデザイン
- ダークモード対応

## 技術スタック

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- PokeAPI

## セットアップ

1. 依存関係のインストール:

```bash
npm install
```

2. 開発サーバーの起動:

```bash
npm run dev
```

3. ブラウザで [http://localhost:3000](http://localhost:3000) を開く

## 使用方法

- トップページでポケモン一覧を表示
- 各ポケモンカードをクリックして詳細情報を表示
- 詳細ページから「一覧に戻る」リンクでトップページに戻る

## ビルド

```bash
npm run build
npm start
```
