# ポケモン図鑑 Webアプリケーション

ポケモンデータを表示するNext.jsベースのWebアプリケーションです。

## 機能

- **ポケモン一覧表示**（第1世代151匹）
  - 検索機能
  - タイプフィルター
  - グリッド表示
- **ポケモン詳細表示**
  - 基本情報（高さ、重さ、性別）
  - タイプ表示
  - 特性表示
  - ステータス表示
  - 画像表示（前向き/後ろ向き、オス/メス、静止画/GIF）
- **ポケモン比較機能**
  - 2匹のポケモンを並べて比較
  - 比較履歴の保存
- **ポケモン診断機能**
  - 簡易診断（5問）
  - 詳細診断（10問）
  - あなたにぴったりのポケモンを診断
- **性格診断機能**
  - ポケモンに基づいた性格診断
- **レスポンシブデザイン**
- **ダークモード対応**

## 技術スタック

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- PokeAPI（ポケモンデータの取得元）

## データソース

このアプリケーションは[PokeAPI](https://pokeapi.co/)から以下の情報を取得しています：

- ポケモンの基本情報（名前、画像、タイプ、高さ、重さなど）
- ステータス（HP、攻撃、防御、特攻、特防、素早さ）
- 特性
- 性別情報

## セットアップ

1. 依存関係のインストール:

```bash
npm install
```

2. 開発サーバーの起動:

```bash
npm run dev
```

3. ブラウザで [http://localhost:3000](http://localhost:3000) を開く

## 使用方法

### ポケモン一覧
- トップページでポケモン一覧を表示
- 検索バーでポケモン名を検索
- タイプフィルターでタイプ別にフィルタリング
- 各ポケモンカードをクリックして詳細情報を表示

### ポケモン詳細
- 基本情報（高さ、重さ、性別）を確認
- タイプと特性を確認
- ステータスを確認
- 画像の種類を切り替えて表示

### ポケモン比較
- `/compare`ページで2匹のポケモンを比較
- ステータスや特性を並べて比較

### ポケモン診断
- `/diagnosis`ページで診断を開始
- 簡易診断または詳細診断を選択
- 質問に答えて、あなたにぴったりのポケモンを診断

### 性格診断
- `/personality`ページで性格診断を実施

## ビルド

```bash
npm run build
npm start
```
